# URL Collector

A Chrome extension that automatically collects URLs with a space between each when you copy them.

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked" and select this extension's folder

## Usage

- Copy any URL in your browser; it will be automatically collected.
- Click the extension icon to view, copy, or clear collected URLs.

## Permissions

- `storage`: Stores collected URLs locally
- `activeTab`: Accesses the current tab to detect URL copies
